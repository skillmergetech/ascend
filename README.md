# Ascend | High-Performance Life Management

Ascend is a high-fidelity personal development command center designed to synchronize micro-actions with macro-vision. Built with Next.js, Tailwind CSS, and Framer Motion for a premium, strategic aesthetic.

## 🚀 Key Features

- **Goal Cascade**: Hierarchical vision tracking (Weekly to Yearly).
- **Mission Control**: Daily task management with priority tiers and morning/evening focus.
- **Financial Fortress**: Strategic wealth allocation based on the 90/10 principle.
- **Performance Hub**: Real-time analytics, momentum gauges, and execution heatmaps.
- **Honor Hall**: Gamified XP and level system with achievement unlocks.
- **Mission Review**: Weekly and monthly tactical retrospectives.

## 🛠 Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS + ShadCN UI
- **State**: React Context API with LocalStorage persistence
- **Animations**: CSS Keyframes + Framer Motion patterns
- **Icons**: Lucide React
- **Charts**: Recharts

## 📦 Deployment to GitHub

### Initial Setup
1. **Create a new repository** on [GitHub](https://github.com/new).
2. **Initialize Git** in your project folder:
   ```bash
   git init
   ```
3. **Stage and Commit** your files:
   ```bash
   git add .
   git commit -m "Initial commit: Ascend v1.2.0"
   ```
4. **Link to GitHub**:
   ```bash
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   ```
5. **Push to Main**:
   ```bash
   git push -u origin main
   ```

### Incremental Updates
When you make new changes (like adding profile uploads or new features), run:
```bash
git add .
git commit -m "Description of your changes"

```
